# Bouldering Tracker（日本語）

このディレクトリには、Googleスプレッドシート用ボルダリングトラッカーのスクリプトとドキュメントが含まれます。

## クイック手順
1. `./update.sh` を実行
2. シートで **Tracker Tools -> Run Post-Update Routine**
3. `Settings!J2` を `EN` または `JA` に設定
4. **Tracker Tools -> Apply UI Language**

## 主要メニュー
- Setup / Repair Spreadsheet
- Run Post-Update Routine
- Rebuild Tables from EventLog
- Import Demo/Test Data
- Reset Data (Safe)
